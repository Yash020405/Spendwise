Spendwise - Product Requirements Document (PRD)

Version: 1.0
Date: January 6, 2026
Author: Yash Agarwal

1. Executive Summary

Spendwise is a modern expense tracking mobile application built with React Native and Expo. The app empowers users to take control of their personal finances through intuitive expense tracking, insightful analytics, and smart budgeting tools.

2. Product Vision

Mission: To simplify personal finance management by providing an intuitive, intelligent, and comprehensive expense tracking solution.

Vision: Become the go-to expense tracking app that helps users achieve their financial goals through smart insights and seamless user experience.

3. Target Audience

Segment Description
Young Professionals (22-35) Tech-savvy individuals managing personal finances
College Students Budget-conscious users tracking daily expenses
Freelancers & Gig Workers Users needing to track business and personal expenses
Families Shared household expense management

4. Core Features (v1.0)

4.1 Expense Management

Feature Description
Add Expense Record with amount, category, payment method, description, date
Edit Expense Modify existing expense records
Delete Expense Remove expenses with confirmation dialog
Built-in Calculator Quick in-app calculation for complex amounts

4.2 Categories

10 predefined categories with visual icons and colors: - Food, Transport, Shopping, Entertainment, Bills - Health, Education, Travel, Subscriptions, Other

4.3 Payment Methods

6 payment options: - Cash, UPI, Credit Card, Debit Card, Net Banking, Wallet

4.4 View Modes & Filtering

  • Daily, Weekly, Monthly expense views
  • Date/period navigation with arrows
  • Category-based filtering
  • Payment method filtering
  • Pull-to-refresh functionality

4.5 Insights & Analytics

  • Pie chart visualization by category
  • Category breakdown with percentages and progress bars
  • Historical trend bar charts (7 days / 6 months / 3 years)
  • Period comparison tool (compare days, months, years)
  • Click on category to filter expenses

4.6 Budget Management

  • Monthly budget setting
  • Carry-over budget toggle
  • Real-time budget tracking
  • Budget alerts at 75%, 90%, and 100% usage

4.7 Settings & Preferences

  • Multi-currency support (INR, USD, EUR, GBP, JPY)
  • Theme options (Light, Dark, System)
  • User profile management

4.8 Offline Support

  • Automatic offline expense saving
  • Auto-sync when connection restores
  • Graceful error handling with informative messages

4.9 Authentication

  • Email/password registration
  • JWT-based secure login
  • Persistent sessions with AsyncStorage
  • Logout functionality

5. Tech Stack

Frontend

  • React Native with Expo
  • Expo Router (file-based routing)
  • Redux Toolkit (state management)
  • AsyncStorage (local persistence)
  • TypeScript

Backend

  • Node.js with Express
  • MongoDB with Mongoose
  • JWT Authentication
  • bcrypt (password hashing)

6. API Endpoints

Authentication

Method Endpoint Description
POST /api/auth/register Register new user
POST /api/auth/login User login

User

Method Endpoint Description
GET /api/users/profile Get user profile

Expenses

Method Endpoint Description
GET /api/expenses Get all expenses
POST /api/expenses Create expense
PUT /api/expenses/:id Update expense
DELETE /api/expenses/:id Delete expense

8. Future Scope (v2.0+)

8.1 Income & Money Addition

  • Income Tracking: Add income entries with source categories (Salary, Freelance, Investments, Gifts, Refunds, Other)
  • Recurring Income: Set up recurring income for regular payments (monthly salary, etc.)
  • Net Balance Display: Show Income - Expenses balance on home screen
  • Income History: View and filter all income entries
  • Savings Rate Tracking: Track percentage of income saved each month
  • Income vs Expense Charts: Visual comparison in insights

8.2 Expense Splitting & Sharing

Core Splitting Features: - Link expenses to contacts from device phonebook - Split expense equally or by custom amounts/percentages - Track per-person share amounts

Payment Status Tracking: - Mark each participant as Paid or Non-Paid - Record payment date when settled - Outstanding balance tracking (who owes whom)

Sharing: - Share expense details via WhatsApp, SMS, Email - In-app notification center for split activity

8.3 Media Attachments

  • Attach photos of receipts/bills to expenses for personal reference
  • View attached media in full-screen gallery
  • Store locally on device (no cloud storage required)

8.4 Additional Features

  • Home screen widgets
  • Local backup and restore
  • Spending pattern analysis (weekday vs weekend)
  • Category-wise spending limits
